About FlipBits

FlipBits turns text into audible encoded audio. The Android app also visualizes playback, waveforms, text encoding, and the binary content behind each character or word in sync; this page is only a quick preview of how the different modes sound.

flash is a deliberately entertainment-first mode: it changes low / high bit Hz, bit duration, and bit spacing to imitate different emotional speaking tones while intentionally ignoring transmission efficiency and noise resistance.

Audio generated by every mode can be decoded in the Android app; for the full experience, download the Android APK from Releases. If you like FlipBits, please give it a star at https://github.com/camellia2077/FlipBits.
