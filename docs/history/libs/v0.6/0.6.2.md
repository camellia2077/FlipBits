## [v0.6.2] - 2026-05-14

### 技术改进/重构 (Changed/Refactor)
* 将 Flash 默认 profile / voicing flavor 的公开 contract 从 `Steady` 统一改为 `Standard`，同步更新 `bag_api`、core config 和 WAV metadata 枚举口径；这次改名不是技术含义变化，而是为了避免 `steady` 在英文里带出“长期且关系稳定的男女朋友”语义，与项目的科技风/科幻风格不匹配，改回更接近“标准说话风格”的 `Standard`。
* 对齐库层与 Android 侧的 Flash style 命名方法，避免 Android XML、UI 文案和 debug/automation 已经写成 `Standard`，而 libs 公开 contract 仍保留 `Steady`，从而在后续 API 消费、WAV metadata 恢复和跨端开发里持续制造歧义。
