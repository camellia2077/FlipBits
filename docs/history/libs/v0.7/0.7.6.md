## [v0.7.6] - 2026-05-30

### 新增功能 (Added)
* 增加 `bag_decode_operation` 生命周期 API，提供 `create` / `pump` / `poll` / `take result` / `cancel` / `destroy` 入口，让 Android 可以用稳定的 native contract 分步 decode 长音频，而不是一次性同步解码。
* 增加 decode operation work plan 和 progress snapshot，公开 `Preparing input`、`Reading PCM`、`Decoding payload`、`Finalizing` 阶段以及 PCM sample 读取进度，用于 Android 保存音频后的 decode 进度条。

### 技术改进/重构 (Changed/Refactor)
* 复用结构化 decode result 复制逻辑，让同步 decode 和 operation decode 输出同一套 text、raw payload、follow timeline、lyrics line、raw segment 与 Ultra frame timeline 数据，支撑 Android 保存到本地后重新打开仍可恢复 visual/tokens。
* 保持 `libs` host 路径走 module-first / `import` 验证，同时把新增 decode operation 作为 `bag_api.h` C ABI contract 暴露给 Android JNI。

### 修复 (Fixed)
* 补充 decode operation ABI 数值、非法参数、增量 pump、取消和结果读取测试，防止 Android 依赖的长音频 decode contract 回归。
