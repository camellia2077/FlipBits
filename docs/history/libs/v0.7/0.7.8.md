## [v0.7.8] - 2026-06-18

### 新增功能 (Added)
* 增加 `bag_get_encode_operation_diagnostics` 接口与 `bag_encode_operation_diagnostics` 结构体，支持获取 flash voicing 编码阶段的各 DSP 模块耗时与样本统计数据。
* 增加 `FlashVoicingStepper` 的诊断性能监控开关 `enable_diagnostics` 参数。
* 补充 `libs/AGENTS.md` 和 `libs/README.md` 中 Voice FX 跨端一致性开发契约，明确以 `bag_apply_voice_fx` 作为离线 canonical API，并规范采样率对齐与进度条显示逻辑。

### 技术改进/重构 (Changed/Refactor)
* 重命名 Voice FX Preset 选项，将 `BAG_VOICE_FX_BINHARIC` 重命名为 `BAG_VOICE_FX_BINARIC_CANT`，并同步更新 core 内核枚举。
* 优化编码与渲染性能，将 `libs/audio_core/src/transport/encode_operation_steps_impl.inc` 中的单次 pump 工作量从 256 提升至 2048 (`kRenderPumpWorkBudget`)，提高吞吐量。
