## [v0.3.3] - 2026-04-23

### 技术改进/重构 (Changed/Refactor)
* 重构播放器播放区的 Kotlin 结构边界，将原本集中在 `AudioPlaybackProgressSection` 里的显示模式、进度时间轴与拖拽动画拆分为独立模块，并把播放器详情页中的 display state 收口为共享状态，减少同一播放区在详情内容与底部控制栏之间重复持有显示逻辑。
* 重构 payload follow 视图的 Kotlin 组织方式，将入口、展示状态、注释模式切换、token 列表、token 卡片与 timeline 纯函数拆到独立文件，降低 `PlaybackDataFollowSection` 的职责密度，方便后续继续演进 `hex/bin` 跟随显示。
* 重构 `LibraryTabScreen` 的页面入口，把本地页面状态、弹窗挂载与列表内容区域分离为独立模块，收口库页面的页面编排职责，避免后续继续把筛选、批量操作和弹窗状态堆回单一入口文件。
