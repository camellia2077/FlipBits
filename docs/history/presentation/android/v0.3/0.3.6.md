## [v0.3.6] - 2026-04-26

### 新增功能 (Added)
* 新增 `Italian`、`Polish` 与风格化语言 `Altum Gothicum` 的 Android 语言选项，并补齐对应的界面文案与样例资源，使设置页可以直接切换到这些新语言。
* 新增 `German`、`Spanish`、`French`、`Italian`、`Russian`、`Traditional Chinese`、`pt-BR`、`Ukrainian`、`Polish` 与 `Altum Gothicum` 的拆分字符串资源，围绕 `Audio / Saved / Settings / About / Validation` 六个职责文件补齐当前 Android 多语言覆盖面。

### 技术改进/重构 (Changed/Refactor)
* 将 Android 英文基线和多语言资源从单一 `strings.xml` 重构为按职责拆分的 `strings_common.xml`、`strings_audio.xml`、`strings_saved.xml`、`strings_settings.xml`、`strings_about.xml` 与 `strings_validation.xml`，统一 `Audio`、播放器、媒体库与设置页的翻译入口。
* 调整语言选择工作流与本地化文档，新增 `android-split-strings-translation-guide.md` 并同步更新 `README.md`、`AGENTS.md` 与本地化规则，明确 split strings 的 source of truth、术语一致性和 Android XML 约束。
* 优化设置页语言选择器为可滚动列表，并扩展 `AndroidSampleInputTextProvider`、语言枚举与语言识别测试，让新增语言与阵营样例文本共用统一的 locale 装载链路。

### 修复 (Fixed)
* 修复 `Polish` 与 `Altum Gothicum` 在设置页选中后不会真正切换应用语言的问题，为 `AppCompat` locale 装载补齐 `pl`、`la` 对应的资源映射与 `localeFilters` 打包声明。
* 修复 `pt-BR`、`Ukrainian` 等新增翻译资源中的 Android XML 引号、撇号和非法转义问题，恢复 `assembleDebug` 资源编译通过。
* 修复多语言样例与语言标签之间的错配风险，确保 `Italian`、`Polish`、`Altum Gothicum` 等新语言切换后会同步加载各自的默认输入样例。
