## [v0.4.5] - 2026-05-08

### 新增功能 (Added)
* 新增 Flash Visual debug-only 性能与运动指标，覆盖 draw/compose 频率、窗口构建、raw playback position、smoother reset、viewport step 与像素跳变，可通过 adb `FlashVisualPerf` 日志进行设备侧诊断。

### 技术改进/重构 (Changed/Refactor)
* 重构 Flash Visual 的播放位置平滑逻辑，让 Canvas 使用 frame-driven visual sample position，并通过最新 raw anchor 做漂移校正，避免长音频播放时跟随播放器粗粒度 sample 更新周期性重启。
* 调整 Flash Visual 窗口请求、窗口刷新间距与 drawable segment 预算，减少长音频播放时重复窗口构建、过量 primitive 绘制和不可见 segment 扫描。
* 整理 Android Flash Visual 架构与诊断文档入口，将 Android 专属架构文档收敛到 `docs/architecture/android/`，并补充 Visual 数据流、adb 抓取、debug 指标解释和卡顿复盘。

### 修复 (Fixed)
* 修复长文本/长音频 Flash Visual 虽然 FPS 较高但仍周期性跳动、闪烁的问题，避免 smoother 随 raw anchor 更新反复 reset。
* 修复 Flash Visual debug overlay 遮挡可视图的问题，将 debug 指标移动到 Visual 画布下方、歌词/bit 文本上方。
