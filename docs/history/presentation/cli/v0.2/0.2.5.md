## [v0.2.5] - 2026-05-14

### 技术改进/重构 (Changed/Refactor)
* 将 Rust CLI 的默认 Flash style、`--flash-style` 帮助文案、FFI profile/flavor 映射和 WAV metadata 恢复口径从 `Steady` 统一改为 `Standard`，让 CLI 与 Android、libs 公开 contract 保持同一套命名方法。

### 修复 (Fixed)
* 修复 Flash 默认 style 在 CLI 与其他产品层之间长期分叉的问题；这次改名的原因不是协议含义变化，而是 `steady` 在英文里容易带出“长期且关系稳定的男女朋友”语义，与项目的科技风/科幻风格不匹配，改为更接近“标准说话风格”的 `Standard` 后，也能避免 Android/XML 已写 `Standard`、而 CLI 仍暴露 `Steady` 导致后续使用和开发混淆。
