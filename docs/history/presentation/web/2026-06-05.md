## 2026-06-05

### 新增功能 (Added)

* 新增 `flash` style 选中说明面板，直接在页面中展示当前 style 的简要描述，并补齐 Web 侧多语言文案。

### 技术改进/重构 (Changed/Refactor)

* 将 `mini` 的 WPM 速度切换从下拉菜单改为横向直接选择，减少一次展开操作并让可选档位更直观。
* 将 `flash` 的 style 切换从下拉菜单改为 `3x2` 直接选择布局，让各个 style 的数量和当前选择状态一眼可见。
* 调整 `flash` style 选项按钮的尺寸和文本换行策略，避免长英文标签被压成竖排拆字。
* 将 `flash` style 说明区域改为组件级响应式布局，按 `flash` 设置区自身宽度决定单列或双列，而不是依赖整页 viewport 断点。
* 调整 `flash` 设置区在桌面布局中的占位方式，让 style 区和说明区在可用横向空间充足时恢复左右分栏，而不是被父级空列压缩成单列手机态。
* 统一 `About FlipBits` 中 `Flash` 的大小写展示，并同步更新高亮逻辑。

### 修复 (Fixed)

* 修复 `flash` style 说明卡片在部分窗口宽度下与 style 选项区发生视觉重叠的问题。
* 修复 `flash` style 区域在不同窗口宽度和预览场景下的中间态布局错位问题，减少桌面浏览器与窄宽容器之间的显示分歧。
