# Trademarks and Unofficial Project Notice

## Purpose

This document explains the trademark and non-affiliation position of the
FlipBits repository.

It is intended to reduce confusion about project origin, sponsorship, and the
limited context in which third-party names may appear in repository materials.

## Independent Project

FlipBits is an independent, unofficial open-source software project.

It is not affiliated with, authorized by, endorsed by, sponsored by, or
officially connected with Games Workshop Limited, Warhammer 40,000, or any
related brand, studio, publisher, or product line.

Nothing in this repository should be interpreted as claiming official status,
licensed status, compatibility certification, publisher backing, or any other
form of commercial or institutional relationship with Games Workshop Limited.

Warhammer, Warhammer 40,000, and certain related names, factions, settings,
characters, and marks are the property of Games Workshop Limited.

All such rights remain with their respective owners.

References to third-party names in this repository are used only in limited
descriptive contexts, such as:

- explaining creative inspiration or tonal direction
- documenting translation context for sample text review
- discussing repository history or prior wording
- clarifying what this project is not affiliated with

These references do not imply ownership, approval, partnership, endorsement,
or authorization.

## Source Code and Repository Materials

The project may contain:

- independent implementation code
- original UI copy
- original sample text
- internal workflow notes for translation or review agents
- documentation that mentions third-party themes or tonal comparisons

Repository documentation or workflow files may occasionally mention external
worldbuilding terms to provide context for translators, reviewers, or agents.
Such contextual references are not part of any claim of official association
and are not intended to present the repository as an official or licensed app.

## No Official Branding Claim

This repository should not be represented as:

- an official Games Workshop product
- an authorized Warhammer 40,000 companion app
- a licensed adaptation
- a publisher-approved utility

Forks, redistributions, and derivative presentations should avoid creating
source confusion or suggesting official endorsement where none exists.

## Project Handling Principle

If repository content appears to create unnecessary confusion, trademark risk,
or misleading association, maintainers may revise names, wording, examples, or
documentation to reduce that risk.

Reports of potentially misleading or inappropriate material are welcome through
the normal project issue channels.
