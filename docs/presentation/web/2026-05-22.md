## [v0.1.0] - 2026-05-22

### 新增功能 (Added)
* 新增 `apps/audio_web` 浏览器演示入口，支持在 GitHub Pages 中直接输入文本、选择传输模式并预览生成的音频效果。
* 新增浏览器侧随机样本文本、长度切换、多语言界面、结果摘要和进度展示，作为 `FlipBits` 在线试听入口。
* 新增 WebAssembly 构建、站点预览、Android sample 文本导出等 Python 工具，并接入 GitHub Pages 部署工作流。

### 技术改进/重构 (Changed/Refactor)
* 复用仓库现有 C++ 编码能力到 Web 演示层，补齐 `audio_web` 的前端职责拆分、WASM bridge 和 Pages 准备流程。
* 按语言拆分 Web i18n 资源，并明确 `Mode`、`Flash Style`、`Morse Speed` 等技术术语的英文保留规则。
* 对齐 Android sample text 资源，支持按语言与 `short` / `long` 长度导出 Web 随机样本。

### 修复 (Fixed)
* 修复本地 Emscripten 环境切换后 Web 构建目录残留旧 toolchain 路径导致的构建失败问题。
* 修复 `mini` 模式输入限制、样本文本切换、播放器结果区布局和 Pages 图标资源路径等预览站点问题。
