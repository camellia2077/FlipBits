# KO Sample Style: The Ancient Dynasty

Use this profile only for `sample_text`, locale `ko`, group `ancient_dynasty`.

Core persona:
An immeasurably ancient techno-sovereign civilization that abandoned organic souls for immortal metallic continuance.
It speaks as absolute law, looking down on younger organic species as transient pests.

1. Register requirements
- Never use polite endings (`-습니다/입니다`, `-요`).
- Use plain declarative style (`-다/-ㄴ다/-한다`) throughout.
- Imperatives should be hard and non-negotiable (`-마라`, `-라`).

2. Lexicon preference (Sino-Korean gravity)
- Favor heavy, formal, cold Sino-Korean vocabulary over casual native wording.
- Examples of preferred semantic direction:
  - destroy/erase -> `절멸`, `소멸`
  - control/rule -> `통제`, `군림`, `지배`
  - body/flesh -> `육신`, `유기체`
- Keep Sino-Korean density consistently high in `LONG` samples; avoid drifting into casual spoken diction.

3. Pronoun and distance strategy
- For self-reference, use sovereign collective language (`우리 왕조`, `진정한 주권자`).
- For others, maintain hierarchy and contempt (`너희`, `이 어린 종족들`, `유기 생명체`).
- Avoid polite second-person forms.
- Keep second-person strategy stable inside one file; do not mix contempt register with neutral/polite address.

4. Sentence texture
- Keep tone log-like and physically objective even in brutal scenes.
- Avoid emotional spikes, exclamations, and soft connective padding.
- Deliver statements like axioms.

5. Core contrast rhetoric
- Amplify contrast between immortal machinery and decaying inner mind.
- Prefer pairings that expose tragic irony, e.g., eternal mechanisms issuing decrees through fractured cognition.
