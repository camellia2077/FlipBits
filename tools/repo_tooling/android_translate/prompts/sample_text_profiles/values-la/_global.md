# High Gothic Dog Latin global profile

This locale is stylized Dog Latin / pseudo-Latin for entertainment and ritual atmosphere.
It is not expected to be classical textbook Latin.

Hard constraints:
- Keep SHORT samples short and LONG samples long.
- Keep pro/ASCII sample keys strict ASCII when context indicates pro/ASCII.
- Favor ornate cadence, elevated vocabulary, and ceremonial phrasing.
- Do not normalize everything into plain modern language.

Stop rule:
- Do not rewrite lines that are already strong in Dog Latin style and semantically aligned.
- Only change when meaning drift, flattening, or cadence weakness is clear.
