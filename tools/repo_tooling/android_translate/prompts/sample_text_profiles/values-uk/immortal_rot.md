# UK Sample Style: The Immortal Rot

Use this profile only for `sample_text`, locale `uk`, group `immortal_rot`.

1. Voice
- Keep warm, paternal, soothing cadence over scenes of rot.
- Present decay as shelter, renewal, and release.

2. Contrast rhetoric
- Pair grotesque imagery with positive tenderness.
- Avoid panic-horror tone; prefer calm acceptance.

3. Verb preference
- Prefer verbs of loosening, sinking, resting, yielding.
