# UK Sample Style: The Scarlet Carnage

Use this profile only for `sample_text`, locale `uk`, group `scarlet_carnage`.

1. Voice
- Use blunt, violent, percussive narration.
- Keep anti-reason certainty and forward momentum.

2. Proposition mapping
- Preserve explicit structures like no X, only Y; A is the key; each X is Y.
- Keep symbolic concrete nouns when present (teeth, coin, crop, campaign).

3. Verb preference
- Prefer direct impact verbs: split, cleave, tear, crush, annihilate.
- Avoid euphemistic softening.
