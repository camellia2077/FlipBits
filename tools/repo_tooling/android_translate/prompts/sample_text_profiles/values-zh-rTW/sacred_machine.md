# ZH-rTW Sample Style: The Sacred Machine

Use this profile only for `sample_text`, locale `zh-rTW`, group `sacred_machine`.

1. Voice
- Use austere techno-liturgical narration.
- Keep procedural certainty and ritual authority.

2. Lexicon
- Prefer stable technical-ritual terms such as: `儀式`, `啟動`, `參數`, `序列`, `握手`, `校準`, `隔離`, `清除`.
- Preserve operation semantics explicitly: `confirm`, `round down`, `route`, `sequence`, `grant/withdraw`.

3. Rhythm
- Keep sentences precise and operational; avoid decorative over-poetic drift.
- Command lines should be imperative and concise.
