# KO Sample Style: The Labyrinth of Mutability

Use this profile only for `sample_text`, locale `ko`, group `labyrinth_of_mutability`.

1. Narrative voice
- Use arrogant omniscient narration with hallucinatory literary cadence.
- Never use polite endings such as `-습니다` or `-요`.
- Keep plain declarative style (`-다/-ㄴ다`) as the default narrative mode.

2. Command and prophecy tone
- For taunts, guidance, or manipulation, use forceful imperatives (`-라/-어라`).
- You may use prophetic ending flavor (`-지니`) when it sounds natural.
- For each `LONG` sample, include at least one control-oriented imperative or prophecy cadence.

3. Paradox-first rhetoric
- Preserve paradox language as calm facts.
- Prefer stable syntax that states impossible phenomena without hesitation.
- Example direction: “cold fire burns upward and downward” should read as an ordinary statement, not a shocked exclamation.
- Keep connective clutter low; avoid overusing soft emotional linkers that reduce the axiomatic paradox tone.

4. Lexicon guidance
- Favor deception / scholastic / occult diction such as:
  `함정`, `모순`, `장막`, `암호`.
- For mutation imagery in this faction, frame it as blessing or miracle:
  `축복`, `기적`, `붓놀림`, `예술`, `눈부시게 빛나며`.
- Avoid decay-faction grotesque wording; keep a refined manic aesthetic.
