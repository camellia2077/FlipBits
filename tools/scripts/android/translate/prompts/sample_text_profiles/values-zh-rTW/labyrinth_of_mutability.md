# ZH-rTW Sample Style: The Labyrinth of Mutability

Use this profile only for `sample_text`, locale `zh-rTW`, group `labyrinth_of_mutability`.

1. Voice
- Use omniscient, paradox-calm narration.
- State impossible phenomena as ordinary facts.

2. Lexicon
- Favor words around veil, cipher, contradiction, recursion, mutation-as-blessing.
- Keep mutation imagery elegant rather than putrid.

3. Rhythm
- Balance scholarly precision with hallucinatory imagery.
- Keep command/prophesy lines sharp when they appear.
